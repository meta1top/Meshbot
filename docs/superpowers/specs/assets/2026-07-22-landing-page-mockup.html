你的数据在哪

不用「安全第一」这种话糊弄你。下面是准确的三档。

留在本机
  • 会话历史与全部消息
  • Agent 的记忆
  • 工具执行记录
  • 工作区里的文件
云端没有对应的表，不做持久化。
过境，但不留存
  • 用云端模型网关时的对话内容
  • 跨设备运行时的执行过程
中转靠内存路由，进程重启即消失。想完全不过境，配置本机模型即可。
存在云端
  • 账号、组织与成员
  • 团队消息
  • 网盘里的文件
  • Agent 的名称与头像
团队协作所必需的部分。模型厂商密钥加密存储，不下发到任何设备。
